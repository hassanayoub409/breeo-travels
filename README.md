# Breeo Travels

A multi-page travel agency website built with pure HTML and CSS — no frameworks, no JavaScript.

![HTML](https://img.shields.io/badge/HTML5-E34F26?logo=html5&logoColor=white)
![CSS](https://img.shields.io/badge/CSS3-1572B6?logo=css3&logoColor=white)

---

## Overview

Breeo Travels is a front-end website for a Pakistan-based travel agency. It features a full multi-page layout with a hero video homepage, destination showcase, services, travel articles, and a contact page — all styled with custom CSS and Font Awesome icons.

---

## Pages

| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Hero video, destinations grid, packages, client logos |
| About | `about.html` | Agency background and team |
| Services | `services.html` | Services offered by the agency |
| Destinations | `destination.html` | Featured travel destinations |
| Articles | `ARTICLES.html` | Travel articles and guides |
| Contact | `contact-us.html` | Contact form and office details |

## Running Locally

No build tools or dependencies needed. Just open the project in your browser:

```bash
# Clone the repository
git clone https://github.com/hassanayoub409/breeo-travels.git
cd Breeo-Travels

# Open in browser
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux
```

Or simply drag and drop `index.html` into any browser window.

## Features

- Hero video background on the homepage
- Destination cards with hover effects
- Services and packages sections
- Travel articles with detail pages
- Contact page with office location, email, and phone
- Consistent navigation and footer across all pages
- Font Awesome icons throughout
- Fully responsive layout
